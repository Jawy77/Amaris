# Task 9: Data Masking vs Data Encryption

## Descripcion
Implementaciones practicas de masking, cifrado y tokenizacion para el portal financiero XYZ.

## Uso
```bash
pip install -r requirements.txt
python masking.py
python encryption.py
python tokenization.py
```

## Archivos
- `masking.py` - Masking estatico y dinamico (Faker, roles)
- `encryption.py` - AES-256-CBC cifrado/descifrado
- `tokenization.py` - Simulacion de vault de tokenizacion PCI-DSS
